We can create animations using a combination of web tools: HTML, CSS and JavaScript.

Here is a simple one: where the purple box is moving inside the big box.

We want the small purple box to go from the left top corner to the right bottom corner.

Have a try ;)

This is the JavaScript code: it calls a function to move the box 


        let timer = setInterval(move, 10)
        let pos = 0;
        let box = document.getElementById("box");
            
        function move () {
            if (pos >= 150)
            {
                clearInterval(timer);
            }
            else {
                pos += 1;
                box.style. = pos+"px";
                box.style. = pos+"px";
            }
        }
